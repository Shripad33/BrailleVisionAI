# BrailleVision AI Documentation

BrailleVision AI is an accessibility-focused AI system designed to recognize physical handwritten or embossed Braille using image upload or camera input and convert it into readable English text and speech in real time. The workflow begins with image acquisition, where the user uploads a Braille image or captures it using a webcam. OpenCV preprocessing techniques such as grayscale conversion, thresholding, noise reduction, and contrast enhancement are applied to improve image clarity and make Braille dots more visible. The processed image is then passed through a YOLOv8-based detection pipeline that identifies Braille dots and extracts their coordinates. These detected dots are grouped into standard 2×3 Braille cells, and each cell is converted into a binary Braille pattern which is mapped to corresponding English alphabets using predefined Braille translation logic. The translated text is displayed on the user interface and further converted into speech using Google Text-to-Speech (gTTS) for accessibility. The technical architecture consists of an HTML, CSS, and JavaScript frontend connected to a Flask backend, integrated with OpenCV image processing, YOLOv8 detection, Braille translation logic, and speech generation modules.
