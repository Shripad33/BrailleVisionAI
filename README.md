# BrailleVision AI

Turning Braille into Text and Speech using AI.

## Features
- Braille image upload
- Braille-to-English translation
- Text-to-speech output
- Accessibility-focused UI
- OpenCV preprocessing
- YOLOv8-inspired detection pipeline

## Tech Stack
- Flask
- OpenCV
- YOLOv8
- HTML/CSS/JavaScript
- gTTS
